Welcome to nginx!

If you see this page, the nginx web server is successfully installed and working. Further configuration is required.

Определите разницу между контейнером и образом: Образ - это файл, содержащий слои, изменяющие файловую систему, а также содержащий рантайм параметры. Контейнер - это процесс, имеющий состояние, экземляр докер образа. Можно ли в контейнере собрать ядро? Нет, нельзя. Контейнер использует ядро ОС, на которой запущен и изменяет файловую систему.

For online documentation and support please refer to nginx.org.
